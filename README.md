# Gamaxy - Gaming Galaxy

Gamaxy is an ASP.NET Core gaming platform where users can:

- Browse games
- Upload games
- Download launchers
- Make payments
- Interact with AI chatbot
- Manage gaming content

## Technologies

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML/CSS/JavaScript
- Bootstrap
